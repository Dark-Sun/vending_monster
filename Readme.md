* Vending Monster

** Running
1. `git clone ..`
2. `chmod +x ./vending_monster`
3. `./vending_monster`

Optionally, to run with pry-byebug

4. `bundle install`
5. `bundle exec ./vending_monster` 

** Running specs

`bundle exec rspec spec/`

** Rubocop
`rubocop`
